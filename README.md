# Meme Generator

## 🧩 Description

Meme Generator is a web application that allows users to create custom memes by adding text to images. Users can select images, input top and bottom text, and generate memes dynamically.

The project focuses on DOM manipulation, real-time updates, and providing an interactive and fun user experience.

## 🚀 Live Demo

https://meme-generator-dusky-zeta.vercel.app/

## 🚀 Technologies

* HTML5
* CSS3
* JavaScript (ES6+)
* Vercel (deployment)

## 🎯 Features

* Select and upload images
* Add top and bottom text to memes
* Generate and preview memes in real time
* Responsive and interactive UI
* Lightweight and fast performance

## 🧠 Key Concepts

* DOM manipulation with JavaScript
* Event handling
* Dynamic text rendering
* Separation of structure, style, and logic
* Real-time updates without page reloads
